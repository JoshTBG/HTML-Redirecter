HTML REDIRECTER
	
	How to use:
			[1] Change The Meta Header Tags url=[CHANGE ME] with your web address of choice.
			[2] Change <a class="button" href="[CHANGE ME]" with your web address of choice.
			[3] Save the file.
			[4] Open / load it on a webserver of choice.
			[5] Kaboom! It Should Redirect you to your web address of choice.
